vimfiles
========

A repo for vimfiles folder modifications.

Installation
============
git clone https://github.com/oscarcardoso/vimfiles.git

Setup Submodules
================
git submodule init && git submodule update

Setup Command-T
===============

	* sudo apt-get install ruby ruby-dev
	* cd ~/.vim/bundle/ruby/command-t/
	* ruby extconf.rb
	* make
